# WINE 150 · 모바일 와인 셀러 (PWA)

와인 **150 품종**과 **이마트 대전복합터미널점 가격대별(1·3·5·10만원) 추천 와인 600병**을
휴대폰에서 오프라인으로 찾아보는 고급 와인리스트 앱입니다.

## 기능
- **품종 검색** · 레드/화이트 분류
- **맛 필터**: 당도·산도·타닌·바디를 `무관/낮음/중간/높음`으로 선택
- **향 필터**: 원하는 향(체리, 자몽, 후추, 미네랄 …)을 고르면 그 향이 **하나라도 포함된** 와인만 표시
- **가격대 필터**: 1·3·5·10만원대
- **마리아주 추천**: 먹을 음식을 입력하면(예: 스테이크, 회, 삼겹살, 치즈) 어울리는 와인을 점수순으로 추천
- **상세 보기**: 품종은 PDF와 같은 틀(구조 막대 · 맛/향 구성 · 고향/유래/산지/마리아주/떼루아/특징),
  와인(병)은 **사진(자동 일러스트, 실제 사진 교체 가능) + 가격대 배지 + 맛·향 · 마리아주 · 구매처 · 2~3줄 설명 + 구조 핀**
- **오프라인 설치(PWA)**: 홈 화면에 추가하면 앱처럼 실행, 네트워크 없이도 동작

## 폴더 구성
```
wine-app/
  index.html      앱 화면
  styles.css      고급 와인리스트 테마
  app.js          검색·필터·마리아주·상세·보틀 이미지 로직
  data.js         150품종 + 600와인 데이터(두 PDF 원본 기반)
  manifest.webmanifest, sw.js, icon-*.png   PWA(설치·오프라인)
  images/         (선택) 실제 와인 사진을 넣는 곳
```

## 실행 방법 (Claude Code / 로컬)
정적 파일이라 서버만 있으면 됩니다. 터미널에서:
```bash
cd wine-app
npx serve .        # 또는: python3 -m http.server 8080
```
브라우저에서 표시되는 주소(예: http://localhost:3000)를 열면 됩니다.
> file:// 로 직접 열어도 대부분 동작하지만, 서비스워커(오프라인)·일부 브라우저 보안 때문에 **로컬 서버 실행을 권장**합니다.

## 휴대폰에 설치하기
1. 무료 호스팅에 올립니다. 가장 쉬운 방법:
   - **GitHub Pages**: 저장소에 `wine-app` 내용을 올리고 Pages 활성화
   - **Vercel / Netlify**: 폴더를 드래그&드롭으로 배포
2. 휴대폰 브라우저로 배포 주소 접속 →
   - iPhone(Safari): 공유 → **홈 화면에 추가**
   - Android(Chrome): 메뉴 → **앱 설치 / 홈 화면에 추가**
3. 한 번 열어두면 오프라인에서도 실행됩니다.

## 실제 와인 사진 넣기 (선택)
기본은 품종/스타일에 맞춘 **자동 보틀 일러스트**가 표시됩니다.
실제 사진을 쓰려면 `images/` 폴더에 파일명을 와인명 슬러그로 저장하세요.
- 슬러그 규칙: 와인 영문명(괄호 앞)을 소문자로, 영숫자 외 문자는 `-` 로.
  - 예) `Frontera Cabernet Sauvignon (프론테라…)` → `images/frontera-cabernet-sauvignon.jpg`
- 파일이 있으면 자동으로 사진이 일러스트 위에 표시되고, 없으면 일러스트가 그대로 보입니다.

## 데이터 갱신
`data.js` 는 `window.WINE_DATA` 한 개의 객체입니다. 품종/와인을 추가·수정하려면 이 파일을 편집하면 됩니다.
(원본은 `와인_포도품종_150선` + `와인_가격대별_추천_150` PDF 데이터에서 생성했습니다.)

## 참고
- 가격대는 정가 기준 근사 밴드이며 할인·환율·입고에 따라 달라질 수 있고, 매장 실시간 재고는 보장하지 않습니다.
- 희귀 품종 일부는 국내 유통이 적어 ‘이마트몰/온라인’ 또는 대표 벤치마크 와인으로 안내합니다.
