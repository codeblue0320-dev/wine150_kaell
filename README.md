# WINE 200 · 이마트 대전복합터미널점 모바일 와인 셀러 (PWA)

매장 매대 영상(2026.07)에서 판독한 **69개 품종·스타일, 총 200병**을
휴대폰에서 오프라인으로 찾아보는 와인리스트 앱입니다.
(wine150_kaell 앱의 형식·디자인을 유지하고 데이터·기능을 확장했습니다)

## 새 기능 / 변경점

- **와인리스트 교체**: 실제 매대 기준 200병 (구매 가능 141병 + 장바구니 비활성 59병)
  - 비활성(결품 추정) 와인은 빨간 **「장바구니 비활성」** 배지로 표시
  - 같은 가격대 복수 추천은 **「세컨드 픽」**, 1순위가 비활성일 때는 **「구매 가능 대안」** 배지
- **맛 필터 중복 선택**: 당도·산도·타닌·바디 각각 `낮음/중간/높음`을 여러 개 선택 가능.
  선택한 것 중 **하나라도 해당하면** 결과에 표시됩니다. (`무관` = 선택 해제)
- **마리아주 TOP 3 추천**: 음식을 입력하면 리스트 안 200병 중
  마리아주 일치도 + Vivino 평점 + 구매 가능 여부로 점수를 매겨 🥇🥈🥉 TOP 3와 차순위를 추천
- **마리아주 필터**: 필터 서랍에 음식 카테고리 12종 —
  선택한 것 중 **하나라도 해당하는** 와인이 있는 품종만 표시
- **실물 사진**: 매장 앱 화면에서 추출·정리한 병 사진 200장 (`images/<id>.jpg`)
- **서빙 가이드**: 와인마다 **최적 시음 온도**와 **추천 와인잔**을 표시
- **상세 정보 확장**: 가격·100ml당 가격·Vivino·별점·원산지(생산자▸밭▸지역▸국가)·등급·세파주·떼루아·양조 스타일
- 카테고리 5분류: 레드 / 화이트 / 스파클링(샴페인 포함) / 디저트 / 로제
- 재고 필터: 전체(비활성 포함) ↔ 구매 가능만

> 참고: "클로드(Claude)가 실시간으로 추천"하는 방식은 정적 GitHub Pages에서는
> API 키가 브라우저에 노출되기 때문에 안전하게 구현할 수 없습니다.
> 대신 위의 로컬 점수 기반 TOP 3 추천 + 마리아주 필터로 대체했습니다.

## 폴더 구성

```
wine-app/
  index.html            앱 화면
  styles.css            테마 (+확장 스타일)
  app.js                검색·필터·마리아주 TOP3·상세 로직
  data.js               69품종 · 200병 데이터 (window.WINE_DATA)
  images/               실물 병 사진 200장 (와인 id.jpg)
  manifest.webmanifest, sw.js, icon-*.png   PWA(설치·오프라인)
```

## 실행 (로컬)

```
cd wine-app
npx serve .        # 또는 python3 -m http.server 8080
```

## GitHub에 올리기

1. GitHub에서 새 저장소 생성 (예: `wine200_emart`)
2. 이 폴더 내용 전체를 업로드:
   ```
   cd wine-app
   git init && git add . && git commit -m "WINE 200 emart cellar"
   git branch -M main
   git remote add origin https://github.com/<계정>/wine200_emart.git
   git push -u origin main
   ```
3. 저장소 Settings → Pages → Branch: `main` / root → Save
4. 몇 분 후 `https://<계정>.github.io/wine200_emart/` 접속
5. 휴대폰: Safari 공유 → 홈 화면에 추가 / Chrome 메뉴 → 앱 설치

## 데이터 갱신

`data.js`는 `window.WINE_DATA` 객체 하나입니다.
`varieties[].tiers`는 `"1만"|"3만"|"5만"|"10만"` 키에 **와인 배열**이 들어가며,
각 와인은 `stock`(Y/F), `role`(main/second/alt), `temp`(시음온도), `glass`(추천잔) 필드를 가집니다.
